safeApiCall

suspend fun <T : Any> safeApiCall(errorMessage: String, leftTries: Byte = 2, call: suspend () -> APIResult<T>): APIResult<T>

Function to make simple safe calls (suspend)

Simple function to make safe calls over (unstable) net Has 2 retries by default, but can be "disabled" by set leftTries to 0. This function uses recursion to to the retries

Parameters

errorMessage - is the message if an error occurs

leftTries - is the number of tries to do on error default is 2

call - is a suspend lambda to be called save
